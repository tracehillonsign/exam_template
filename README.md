# Шаблон десктоп-приложения для квалификационного экзамена

Данный шаблон предназначен для того, чтобы убрать рутинные задачи при подготовке к экзамену. Да и какой смысл тратить время постоянно на написание одного и того же? Вот именно.

### Содержание шаблона
- `main.py` — точка входа в программу.
- `config.ini` — конфигурационный файл, настраивает подключение к БД и QTableWidget.
- `src/database.py` — подключение и работа с БД.
- `src/interface.py` — графическая состовляющая приложения.
- `src/utils.py` — дополнительный функционал для работы шаблона.

### Работа с шаблоном
1. Установить любым удобным способом `setup.py`.
2. Установка самого шаблона.
```bash
python setup.py
```
3. Настройка конфигурационного файла `config.ini`.
4. Написание кода в `database.py` и `interface.py`.

### Дополнительно
В `setup.py` присутсвтует автоматическая установка зависимостей(PyQt6, PyMySQL). Для чистой установки используйте флаг `--no-deps`.
```bash
python setup.py --no-deps
```
В случае использования шаблона не на устройствах колледжа рекомендуется чистая установка с использованием виртуального окружения Python. Изначально скрипт ставит зависимости глобально, поскольку в колледже проблема с доступом к виртуальному окружению.

> По мере доработки скрипта планируется добавление примеров вызовов функций (запросы, хранимые процедуры (с параметрами и без) и т.д.) в database.py. Примеры будут добавлены именно в файл репозитория, а не шаблона.
